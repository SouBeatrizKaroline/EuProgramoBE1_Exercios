Não se pode utilizar as palavras reservadas do que as já pre estabelecidas

O JavaScript distingue letras maiuscula e minuscula

Tudo é em inglês



Como declarar uma função?

Usamos a palavra reservada function

Podemos dar um identificador

Não esquecer de usar os parenteses

Dentro das chaves colocamos as instruções

O identificador pode ser qualquer um, porém é super importante usar verbos e descrever o que ele faz

Não se preocupe se os identificadores ficarem grandes, desde que sejam descritivos o suficiente

Use também o camelCase, quando usamos a segunda palavra com a primeira letra maiuscula. Assim fica mais fácil de ler.

Para chamar uma função é só escrever o nome dela e colocar ()
